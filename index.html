隐私政策

本应用（“图片生成”）尊重并保护用户隐私。本政策说明我们如何处理您的数据。

1. 数据使用

  • 您选择的图片通过 HTTPS 加密传输至 Fal.ai 服务器进行 AI 处理。
  • 图片仅用于模型推理与生成优化结果。
  • 根据 Fal.ai 隐私政策，上传图片将在处理完成后自动删除。
  • 本应用不收集或关联个人身份信息。
  • 不建立用户账户系统，不进行广告跟踪或行为分析。

2. 内容安全

  • 启用 NSFW（不当内容）检测机制。
  • 违规内容将无法生成或保存。

3. 本地存储

  • 不会在服务器端长期存储图片或处理历史。
  • 生成结果仅保存在设备本地沙箱环境。
  • API 密钥安全管理，生产版本不向用户暴露。

4. 版权与责任

  • 用户对上传内容合法性与版权承担责任。
  • 本应用仅提供处理工具，不对结果质量作保证。
